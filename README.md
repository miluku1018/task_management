## 部署步驟
1. brew tap heroku/brew && brew install heroku 安裝Heroku in macOS
2. heroku login 登入Heroku
3. git status 確保git狀態是乾淨的
4. heroku create 在Heroku開一個伺服器
5. git remote -v 可以看到有hook
6. git push heroku master 推上去Heroku
